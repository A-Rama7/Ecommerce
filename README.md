# Projet:

Projet de groupe de fin de première réalisé en méthode agile SCRUM.
Technologies utilisé: 
-React JS
-Symfony
-Git
-MYSQL


# Pour démarrer le projet:
 télécharger les dépendances dans le back:

### composer i

lancer le back:
### symfony serve

télécharger les dépendances dans le front:
### npm i

lancer le front:
### npm start
